# Dairy-Farm-Shop-Management-System
The importance of information and efficient information management is steadily increasing due to the evolution of new technologies and high-capacity storage media but also because growing market dynamics raise information needs. A marketing decision support system (MDSS) can be of particular importance as it supports organizations in collecting, storing, processing, and disseminating information, and in the decision-making process by providing forecasts and decision models. The following article provides insights into a successful implementation of a MDSS in dairy industry.
